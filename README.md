# joguinho-forca
Este projeto é uma aplicação front-end desenvolvida com HTML, CSS e javaScript.
O objetivo é praticar e demonstrar conceitos fundamentais de desenvolvimento web.


## Tecnologias Utilizadas
- HTML -> estrutura das paginas
- CSS -> estilização e layout
- JavaScript -> interatividade e lógica
